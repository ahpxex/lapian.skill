# 喜剧 (Comedy)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 笑点结构

- **笑点类型**: 语言幽默 / 情境喜剧 / 肢体喜剧 / 反差 / 荒诞 / 黑色幽默？
- **铺垫与回收**: 笑点的铺垫长度？callback 的使用？
- **密度与分布**: 笑点的频率？密集轰炸 vs 留白蓄力？
- **笑点层级**: 是否有多层笑点（表面笑点 + 深层讽刺）？

## 2. 节奏与 Timing

- **comic timing**: 停顿、延迟、加速的精确控制？
- **剪辑节奏**: 剪辑如何服务于笑点的 timing？cut 的时机？
- **反应镜头**: 反应镜头对笑点的放大作用？

## 3. 喜剧类型学

- **讽刺 (Satire)**: 讽刺对象是什么？讽刺是否锋利？
- **闹剧 (Farce)**: 误会、身份混乱、失控局面的设计？
- **浪漫喜剧**: 笑点与情感线的平衡？
- **黑色喜剧**: 幽默与残酷的边界在哪？
- **荒诞喜剧**: 荒诞的内在逻辑是什么？

## 4. 角色喜剧

- **喜剧人格**: 角色的喜剧性来源——缺陷、执念、反差？
- **ensemble dynamics**: 群戏中角色的喜剧功能分配？
- **角色弧线**: 喜剧角色是否有真正的成长？还是为了喜剧效果保持不变？

## 5. 社会功能

- **笑的对象**: 笑点是向上嘲讽（权力者）还是向下嘲笑（弱者）？
- **禁忌处理**: 是否触碰敏感话题？处理方式是否有效？
- **解构功能**: 喜剧是否在解构某种社会规范或类型惯例？
