# 音乐剧/歌舞 (Musical)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 歌舞段落的叙事功能

- **功能类型**: 歌舞段落是推进情节 / 表达内心 / 纯粹娱乐 / 世界观展示？
- **进出歌舞的衔接**: 从对白到歌舞的过渡是否自然？过渡设计？
- **信息增量**: 歌舞段落结束后，叙事/角色有没有前进？还是原地打转？
- **歌词叙事**: 歌词在传递什么叙事信息？是否有潜台词？

## 2. 情感升格机制

- **情感阈值**: 什么时候情感强度溢出到非唱不可的地步？这个阈值是否可信？
- **升格节奏**: 从对白到演唱到舞蹈的情感升级？
- **音乐的情感编码**: 旋律、和声、节拍如何编码不同情感？
- **高潮曲目**: 全片的 showstopper 如何在情感和技术上超越其他段落？

## 3. 编舞与镜头配合

- **舞蹈可见性**: 镜头是否完整呈现了舞蹈动作？还是被快切碎片化？
- **摄影机运动**: 摄影机是配合舞蹈运动还是独立运动？
- **空间编排**: 舞蹈如何利用和重新定义空间？
- **群舞调度**: 大规模群舞的空间调度和视觉层次？

## 4. 音乐设计

- **曲风选择**: 音乐风格与故事设定/时代的关系？
- **主题旋律**: 是否有贯穿全片的主题旋律？其变奏如何跟踪叙事？
- **reprise 的使用**: 旋律重现时语境变化带来的意义变化？
- **diegetic vs non-diegetic**: 音乐在故事世界内是否存在？角色是否"知道"自己在唱歌？

## 5. 类型历史位置

- **传统与创新**: 相对于音乐剧/歌舞片传统（黄金时代、摇滚音乐剧、hip-hop 音乐剧），这部作品的位置？
- **舞台到银幕**: 如果改编自舞台剧，电影化的策略是什么？
- **当代音乐剧的挑战**: 如何处理当代观众对"突然唱歌"的不适感？
