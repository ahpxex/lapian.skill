# 心理/精神 (Psychological)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 主观视角可靠性

- **感知扭曲**: 观众看到的是客观现实还是角色的主观感知？
- **可靠性信号**: 创作者通过什么线索暗示视角不可靠？
- **真相层级**: 是否存在多层"真相"？最终是否给出确定答案？
- **观众的认知位置**: 观众被放在什么认知位置？共谋者/被欺骗者/旁观者？

## 2. 内心外化手法

- **视觉外化**: 心理状态如何通过视觉元素（色彩、构图、空间）被外化？
- **声音外化**: 内心声音、幻听、环境音变形如何表达心理状态？
- **环境映射**: 外部环境是否随角色心理状态变化？
- **身体症状**: 心理状态通过身体表现（颤抖、失眠、食欲变化）的呈现？

## 3. 现实/幻觉边界

- **边界模糊的手法**: 现实和幻觉如何被混淆？剪辑/镜头语言的处理？
- **观众锚点**: 观众是否有可靠的"现实锚点"？还是完全失去参照？
- **揭示策略**: 幻觉/妄想何时被揭穿？揭穿方式是突然还是渐进？

## 4. 心理机制

- **具体病理**: 如果涉及具体心理状况，呈现是否准确且尊重？
- **创伤叙事**: 创伤如何被叙事化？闪回/碎片/压抑/重现？
- **防御机制**: 角色的心理防御机制如何在行为和叙事中体现？
- **治愈/恶化弧线**: 心理状态的变化轨迹？

## 5. 观众心理操控

- **认知失调**: 创作者如何制造观众的认知失调？
- **共情与距离**: 观众在多大程度上被邀请进入角色的主观世界？
- **不适感的功能**: 有意制造的不适感服务于什么目的？
