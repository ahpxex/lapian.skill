# 末日/后启示录 (Post-Apocalyptic)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 文明废墟美学

- **废墟景观**: 文明遗迹如何被视觉化？废墟的美感来自哪？
- **自然回归**: 自然侵蚀人造物的视觉处理？
- **旧世界遗物**: 末日前的日常物品在新世界中的叙事功能？（怀旧、讽刺、实用）
- **时间的视觉化**: 灾难过去了多久？视觉设计如何传达时间距离？

## 2. 生存伦理

- **道德底线**: 极端环境下道德底线如何被重新定义？
- **资源稀缺**: 资源争夺如何驱动冲突？分配正义的问题？
- **暴力正当性**: 为了生存的暴力在多大程度上被叙事正当化？
- **人性测试**: 极端环境是揭示人性本质还是扭曲人性？

## 3. 社会重建叙事

- **新秩序**: 末日后产生了什么样的社会组织？权力来源是什么？
- **文明 vs 野蛮**: 新社会在文明-野蛮光谱上的位置？
- **旧制度的幽灵**: 末日前的社会结构（阶级、性别、种族）是否在新世界重现？
- **希望的形态**: 重建的希望以什么形式出现？是否可信？

## 4. 灾难起源叙事

- **灾难类型**: 核战 / 疫病 / 气候 / AI / 外星入侵？选择映射什么当代焦虑？
- **因果链**: 灾难的因果是否被交代？归因于人为还是天灾？
- **灾难的不可逆性**: 是否有"修复"的可能？不可逆性如何影响叙事基调？

## 5. 类型惯例

- **公路叙事**: 旅程/迁徙作为叙事结构？
- **据点叙事**: 防守/建设作为叙事结构？
- **孤独者叙事**: 独自生存的心理状态？
- **类型混合**: 与恐怖、动作、科幻等其他题材的混合方式？
