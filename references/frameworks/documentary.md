# 纪录片 (Documentary)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 真实性建构

- **纪录片模式**: 观察式 / 参与式 / 解说式 / 反身式 / 诗意式 / 表演式？
- **干预程度**: 创作者对拍摄对象的干预有多大？是否透明？
- **搬演与重构**: 是否有搬演（reenactment）？处理方式是否诚实？
- **素材选择偏差**: 素材的选择和排列如何构建了一个特定的"真实"？

## 2. 叙事立场

- **创作者位置**: 创作者是隐身还是在场？在场的程度？
- **对象关系**: 创作者与拍摄对象的权力关系？是否有剥削风险？
- **立场透明度**: 创作者的立场是否被明示？还是伪装成"客观"？
- **多声部 vs 单一叙事**: 是否呈现了多个视角？反对声音的空间？

## 3. 素材编排伦理

- **时间线操控**: 素材的时间顺序是否被重新排列？排列的叙事目的？
- **并置效果**: 不同素材的并置如何创造新的意义？
- **配乐引导**: 音乐是否在引导观众的情感判断？
- **采访剪辑**: 采访素材的剪辑是否改变了原意？

## 4. 信息架构

- **信息密度**: 信息的投放节奏和密度？
- **复杂性管理**: 复杂议题如何被分解和呈现？是否过度简化？
- **数据可视化**: 图表、动画、文字覆盖的使用？
- **专家与普通人**: 权威声音和个人经验的平衡？

## 5. 社会功能

- **行动召唤**: 纪录片是否试图推动特定行动或改变？
- **社群影响**: 对拍摄对象/社群的实际影响？拍摄过程本身的影响？
- **持久性**: 纪录片捕捉的瞬间在时间中如何获得新的意义？
