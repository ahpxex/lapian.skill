# 战争 (War)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 战争的道德叙事

- **立场**: 反战 / 中立呈现 / 英雄主义？是否有明确的道德判断？
- **敌人的呈现**: 对手是被人性化还是符号化？去人性化的程度和手法？
- **服从与反抗**: 个体在军事体制中的道德选择如何被呈现？

## 2. 暴力呈现伦理

- **战斗场景的写实程度**: 追求真实还是有选择性地过滤？
- **死亡的重量**: 每一次死亡是否被赋予叙事重量？还是变成背景噪音？
- **创伤的呈现**: PTSD、心理创伤、道德伤害是否被认真处理？
- **壮观化风险**: 战争场面是否被拍得"好看"到消解了反战意图？

## 3. 个体 vs 集体

- **个人故事与历史事件的关系**: 个体叙事如何嵌入宏大历史？
- **士兵的主体性**: 角色是有自主意志的个体还是棋子？
- **战友关系**: 极端环境下的人际关系如何被塑造？

## 4. 空间与身体

- **战场空间**: 空间的混乱/清晰如何反映战争体验？
- **身体的脆弱性**: 身体在战争中的物理脆弱如何被呈现？
- **感官轰炸**: 声音设计如何模拟战争的感官过载？

## 5. 历史与记忆

- **历史准确性**: 对历史事件的还原程度？虚构与史实的边界？
- **记忆政治**: 这部作品参与了什么样的战争记忆建构？
- **当代映射**: 历史战争叙事是否映射当代议题？
