# Vlog

叠加在 `_base.md` 之上的题材特化分析维度。分析重心从"为什么好"转向"为什么有效"。

## 1. 人格化策略

- **人设构建**: 创作者呈现的人格是什么？与真实人格的距离？
- **一致性**: 人设在不同视频/场景间是否一致？
- **独特性**: 人格化的辨识度？与同类创作者的区分？
- **亲密感制造**: 通过什么手法让观众产生"我认识这个人"的感觉？

## 2. 生活节奏 vs 剪辑节奏

- **时间压缩**: 一天/一段时间的生活如何被压缩成几分钟？
- **选择性呈现**: 选择展示什么、隐藏什么？选择标准是什么？
- **真实感 vs 精致度**: 粗糙手持 vs 精心构图的平衡？
- **日常的叙事化**: 普通日常如何被赋予叙事弧线？

## 3. 剪辑风格与人设一致性

- **剪辑风格**: 快节奏跳切 / 舒缓长镜头 / 蒙太奇 / 混合？
- **风格与人设的匹配**: 剪辑风格是否强化了人格表达？
- **音乐选择**: BGM 风格与内容/人设的关系？
- **文字/图形**: 字幕、贴图、特效的使用风格？

## 4. 叙事钩子

- **开头策略**: 日常 vlog 如何制造观看理由？
- **悬念维持**: 是否有"今天要完成的事"或"结果如何"的悬念线？
- **情感弧线**: 一支 vlog 内是否有情绪的起承转合？
- **结尾设计**: 是否有清晰的结束感？还是自然消散？

## 5. 传播机制

- **可分享性**: 观众看完后的分享动机是什么？
- **系列化**: 是否有系列 vlog 的连续性设计？
- **社群建设**: 内容是否在培养固定观众群？如何做到的？
- **平台适配**: 内容是否针对特定平台优化（YouTube / B站 / 抖音）？
