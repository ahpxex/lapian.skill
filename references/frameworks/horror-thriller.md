# 恐怖/惊悚 (Horror/Thriller)

叠加在 `_base.md` 之上的题材特化分析维度。

## 1. 恐惧构建机制

- **恐惧类型**: jump scare / 心理恐惧 / 身体恐惧 / 存在恐惧 / 不可名状恐惧？
- **恐惧曲线**: 恐惧感的积累和释放节奏？持续压迫 vs 间歇释放？
- **恐惧来源**: 恐惧来自外部威胁还是内在崩溃？已知恐惧 vs 未知恐惧？

## 2. 信息不对称

- **观众 vs 角色**: 观众知道危险但角色不知道（悬念）？还是双方都不知道（surprise）？
- **显示 vs 暗示**: 威胁被直接展示还是暗示？不可见的恐惧如何维持？
- **信息揭露节奏**: 真相在什么时候、以什么方式被揭示？

## 3. 氛围营造

- **空间设计**: 封闭空间 / 空旷空间 / 日常空间的恐怖化？
- **声音的恐惧功能**: 静默、低频、突发音效的部署策略？
- **光影的恐惧功能**: 黑暗、阴影、反常光源的使用？
- **节奏操控**: 缓慢推进制造不安 vs 突然加速制造惊吓？

## 4. 身体与暴力

- **暴力的呈现选择**: 直接展示 / 暗示 / 画外处理？选择背后的美学立场？
- **身体恐惧**: 变形、侵入、失控——身体作为恐惧载体？
- **gore 的叙事功能**: 血腥是服务于恐惧还是沦为猎奇？

## 5. 类型自觉

- **类型规则的遵守与打破**: 是否利用观众对恐怖片规则的预期来制造效果？
- **元叙事**: 是否有对类型本身的自反？（如 Scream 式的类型自觉）
- **恐惧的社会映射**: 恐惧的具体形态映射了什么集体焦虑？
